# Deals for Devs

An application to send upcoming and ongoing deals straight to your inbox every month!

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Xata](https://xata.io/)
- [Sentry](https://sentry.io/)
- [Clerk](https://clerk.com/)
